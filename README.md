# 🚀 Crypto Sentiment vs Trader Behavior Analysis

## 📌 Overview
This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance using historical trading data. The goal is to uncover patterns that can help improve trading strategies.

---

## 🧠 Key Objectives
- Analyze trader profitability under different market sentiments
- Evaluate win rates and trading behavior
- Identify risk patterns (volatility)
- Derive actionable trading strategies

---

## 📂 Dataset
- Bitcoin Fear & Greed Index (Sentiment Data)
- Historical Trader Data (Hyperliquid)

---

## ⚙️ Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn

---

## 🔍 Analysis Performed
- Profitability vs Sentiment
- Win Rate Analysis
- Trade Count Analysis
- Long vs Short Performance
- Volatility (Risk) Analysis

---

## 📊 Key Insights
- Highest profit observed during **Greed**
- Win rate highest in **Extreme Greed**, lowest in **Neutral**
- Volatility is highest during **Greed and Fear**
- **Short trades perform best in Greed**
- **Long trades perform better in Fear**
- Most trading activity occurs during **Fear**

---

## 📈 Strategy Recommendations
- Avoid trading in Neutral markets
- Use short positions during Greed
- Use long positions during Fear
- Manage risk during high volatility periods
